# v1.0 — Monetization & Additional Features

**Phase:** v1.0 Full Launch (Q3 2026)

## Monetization

- [ ] 14-day free trial, no credit card required
- [ ] $24.99/year or $2.99/month subscription
- [ ] iOS: App Store billing integration (StoreKit 2)
- [ ] Android: Google Play Billing integration
- [ ] Single tier — all features included
- [ ] Viewer access (for future shared maps) always free
- [ ] Trial expiration handling and upgrade prompts
- [ ] Restore purchases flow
- [ ] Receipt validation on server

## Timeline View (History)

- [ ] Chronological view of all interactions across all relationships
- [ ] Show both manually logged and auto-detected interactions
- [ ] Filter by person
- [ ] Filter by category/circle
- [ ] Filter by interaction type (call, email, in-person, etc.)
- [ ] Infinite scroll with pagination
- [ ] Diary-like display: "When did I last see X?"

## Custom Circles

- [ ] Create new custom circles with user-defined name
- [ ] Choose custom color for circles
- [ ] Rename existing circles
- [ ] Delete custom circles (reassign members)

## Home Screen Widgets

- [ ] iOS (WidgetKit):
  - [ ] Small widget: 1 person to reconnect with (photo + name + time since last contact)
  - [ ] Medium widget: top 3 reconnection suggestions with quick-action buttons
  - [ ] Large widget: mini People Feed with 5 entries, tappable to open app
- [ ] Android (Glance):
  - [ ] Small widget
  - [ ] Medium widget
  - [ ] Large widget
